# M2 : Myeongjin is Crazy (Racing) 2

학교 축제용 1v1 온라인 레이싱 게임의 기획·구현 기준 문서다. 트랙은 3D 메시, 차량·아이템은 카메라를 향하는 빌보드 스프라이트로 렌더링하는 2.5D 게임이다.

## 문서 안내

이 파일은 핵심 규칙만 유지한다. 상세 작업 이력, 검증 방법, 현재 구현 상태와 이슈는 [`Instructions/`](Instructions/README.md)에서 관리한다.

- [현재 상태·31차 작업](Instructions/CURRENT_STATUS.md)
- [개발·검증·커밋 절차](Instructions/DEVELOPMENT_WORKFLOW.md)
- [알려진 이슈·실제 화면 확인](Instructions/KNOWN_ISSUES.md)

## 기술 스택

- Unity 6.3 LTS, C#
- Unity Netcode for GameObjects + Unity Transport
- Unity Multiplayer Services: Lobby + Relay 방 코드, 익명 인증
- 1v1 온라인 레이스. 호스트는 레이스 상태·아이템 스폰·공격 판정을 권위적으로 처리하며 이동은 오너 권위로 복제한다.

## 개발 우선순위

1. 로컬 코어 루프: 차량 조작·트랙·체크포인트·바퀴·타이머
2. 아이템: 스폰·슬롯·사용·피격 효과
3. 스테이지 게이지
4. 온라인 1v1 검증
5. UI 실제 구현, 나머지 스테이지/승리조건/아트 확장

## 현재 핵심 상태

- 로컬 코어 루프와 세 스테이지의 게이지·해저드·별점이 구현됐다.
- NGO Milestone 1·2a·2b는 실제 2인 테스트를 통과했고, Milestone 3 Relay/Lobby 방 코드도 코드·자동 검증까지 완료됐다.
- 13종 아이템 전체 로스터와 안전한 스프라이트 12종이 구현됐다. 황금방패는 오리지널 금색 표현을 쓴다.
- `VictoryCondition.SimpleFinish`와 `VictoryCondition.StarBet`이 구현됐다.
- 정식 HUD와 스피드전 방 설정은 구현됐다. 남은 우선순위와 검증 범위는 `Instructions/CURRENT_STATUS.md`를 따른다.

## 게임 모드

- **아이템전**: 트랙에 랜덤 스폰되는 가속·공격·방어 아이템으로 대결한다. 인벤토리는 주/보조 2칸이며, 주 → 보조 순서로 채운 뒤 둘 다 차면 새 아이템이 주 슬롯을 교체한다.
- **스피드전**: 5바퀴·단순 완주·최대 100km/h로 고정되며, 레이스 중 기본 휘발유가 자동 지급된다. 호스트는 방 설정 UI에서 아이템전과 전환할 수 있다.

## 바퀴·승리 조건

- 바퀴 수: 1, 3(기본), 5. 바퀴 수가 많을수록 난이도가 높다.
- **단순 완주**: 목표 바퀴 수를 먼저 완주하면 승리.
- **별점 내기**: 완주자 전체의 별점(최대 6★)을 비교하고, 동점이면 완주시간이 빠른 쪽이 승리한다. 아무도 완주하지 못하면 무승부이며 별점과 시간까지 같으면 무승부다.

## 경기 흐름

1. 조작법 안내
2. 3초 카운트다운
3. 1바퀴째 제한시간 180초 시작
4. 2/3/4바퀴 완주 시 각각 +45/+30/+15초
5. 제한시간 내 승리 조건 판정
6. 결과 화면(순위·통계·별점)

## 아이템 시스템

### 공통 규칙

- 스폰 시 가속/공격/방어 계열을 균일하게 고른다. 그 순간 10% 확률로 해당 계열의 파생형 풀에서 가중치 추첨한다.
- 파생형 가중치(계열 풀 안의 합계 100): 가속은 슈퍼 휘발유 60·해피버스데이투유 28·재석 유 12, 공격은 C4 23·다이너마이트 28·막대형 수류탄 22·원자폭탄 4·러브레터 23, 방어는 가시 방패 72·황금방패 28이다.
- 공격 피격 시 즉시 멈춘 뒤 정상 가속도로 재가속한다.
- 막대형 수류탄의 근접 감지 거리는 **5m**, 폭발 반경은 10m다.
- 기본 폭탄을 150초간 보유하면 원자폭탄으로 승급한다. 충전 중 피격되면 즉시 원자폭발한다.

### 가속 아이템 (Ctrl)

| 아이템 | 지속 | 최대 가속도 |
|---|---:|---:|
| 휘발유 | 2.0초 | +20km/h |
| 슈퍼 휘발유 | 2.0초 | +35km/h |
| 해피버스데이투유 | 4.0초 | +0km/h |
| 재석 유 | 1.0초 | +100km/h |

### 공격 아이템 (E, C4 기폭은 P)

| 아이템 | 동작 |
|---|---|
| 폭탄 | 3초 후 반경 5m 폭발 |
| C4 | P로 수동 기폭, 반경 8m |
| 다이너마이트 | 4초 후 반경 10m 폭발 |
| 막대형 수류탄 | 상대 5m 접근 시 반경 10m 폭발 |
| 원자폭탄 | 즉시 반경 10km 판정, 경기 무승부 종료 |
| 명진이의 러브레터 | 폭탄과 동일, 피격 하트 효과 |

### 방어 아이템 (E)

- 방패: 4초, 폭탄·러브레터 1회 방어.
- 가시 방패: 4초, 폭탄·러브레터·다이너마이트 1회 방어. 다이너마이트는 반사한다.
- 황금방패: 1초, 원자폭탄을 제외한 모든 공격 1회 방어.
- 아이콘·플레이버 텍스트는 반드시 오리지널로 만든다. 실존 국가 문장이나 정치풍자를 쓰지 않는다.

## 스테이지

모든 게이지는 `StageGaugeSystem`을 상속한다. 게이지 구현을 복제하지 않는다.

### 비키니시티

- 산소 100, -2/초, 숨방울로 30% 회복. 고갈 후 30초 유예가 지나면 게임오버.
- 피격·지형지물·벽 충돌은 비법을 떨어뜨린다.
- 목표: 비법 놓침 10/3/0회 이하. 추가목표: 2:30/2:00/1:45 이내 완주.
- 스폰지밥 IP 대체 여부는 **미결정**이다. 이 결정을 임의로 바꾸지 않는다.

### 아프리카TV

- 멘탈은 팬 응원 존에서 회복하며, 패시브 변화는 0/초다.
- 공격 피격: 별풍선 1개 손실 + 멘탈 +20. 방송사고 존: 멘탈 +15, 10초 조향 반전.
- 멘탈 최대 시 2초 조작 불가.
- 목표: 별풍선 놓침 15/5/0회 이하. 추가목표: 3:15/2:50/2:30 이내 완주.

### 네더 요새

- 체온 기본 +1/초, 오아시스에서 회복, 80% 경고, 100% 즉시 화상 게임오버.
- 일반 피격 +15, 용암 근처 피격 +40, 용암 체류 +10/초.
- 가스트 파이어볼은 코스 밖으로 튕겨낸다.
- 목표: 화상 경고 5/2/0회 이하. 추가목표: 1:20/1:10/1:00 이내 완주.

## 코드 컨벤션

- 네임스페이스: `M2.Core`, `M2.Player`, `M2.Items`, `M2.Stage`, `M2.Network`, `M2.UI`
- 게이지는 항상 `StageGaugeSystem`을 상속한다.
- 조정 가능한 수치는 Inspector에 노출하는 public 필드로 둔다.
- 네트워크 아이템 ID는 명시적 `NetItemId`를 사용한다. 산술적인 tier→ID 매핑을 다시 도입하지 않는다.
