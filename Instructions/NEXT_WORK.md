# 다음 작업 메모

기준일: 2026-07-13

## 다음 작업 — UI 실제 구현

기준 디자인은 사용자가 제공한 `M2 Racing UI.html`이다. 현재의 코드 기반 HUD·방 설정은 기능 확인용 임시 표현이므로, 다음 작업에서는 HTML의 화면 구성·색·타이포그래피·여백·버튼 상태를 Unity UI로 실제 반영한다. 단순히 여섯 화면을 그리는 데서 끝내지 않고, 아래 데이터와 네트워크 흐름까지 실제 게임 기능에 연결한다.

### 필수 화면

1. **메인** — 시작 진입, 방 만들기·방 참가하기, 아바타 설정, 환경설정, 배경과 CTA를 HTML 메인 화면과 동일하게 구현한다.
2. **로비** — 크게 표시되는 방 코드, 참가자 대기/준비 상태, 방장 전용 게임 설정을 구현하고 현재 Relay 기능에 연결한다. 방 만들기와 방 참가하기의 진입 흐름도 구분한다.
3. **인게임** — 바퀴·시간·순위/상대 상태·스테이지 게이지·아이템 정보를 HTML 인게임 화면과 동일한 계층과 크기로 구현한다. 아이템전은 슬롯을 표시하고, 스피드전은 휘발유 슬롯을 표시하지 않으며 5초 자동 지급과 부스터 피드백을 표시한다.
4. **결과** — 순위·완주 시간·별점·승리 조건·아바타/이름을 실제 결과 데이터로 채운다. 다시 하기·로비로·메인으로 버튼도 호스트와 참가자 모두에게 실제로 동작해야 한다.
5. **아바타** — 몸 색, 눈, 입, 볼, 귀, 모자, 이름, 번호판을 선택·미리보기·저장하고 메인·로비·HUD·결과에 반영한다.
6. **설정** — 마스터/BGM/SFX 볼륨, 그래픽 품질, 전체 화면/창 모드, 언어를 HTML 설정 화면과 동일하게 구현하고 실제 설정값으로 저장·적용한다.

### 구현 선행 조건과 연결 범위

1. **공통 UI 흐름** — 메인 → 방 만들기/참가 → 로비 → 레이스 → 결과 → 로비/메인 및 아바타·설정 진입/복귀를 단일 UI 흐름으로 구성한다. 기존 기능 확인용 HUD와 중복 표시되지 않도록 교체 또는 통합 범위를 명확히 한다.
2. **로비 설정 동기화** — 게임 모드, 바퀴 수, 스테이지, 승리 조건은 방장만 변경하고 참가자에게 동기화한다. 비키니시티·아프리카TV·네더요새 스테이지 선택은 현재 개발용 전환에만 연결되어 있으므로, 룸 설정 기반 선택·로드·동기화를 구현한다.
3. **프로필·설정 영속화** — 아바타와 환경설정은 로컬에 저장하고, 다음 실행 및 각 화면에서 동일한 값으로 복원한다.
4. **온라인 결과 동기화** — 현재의 승자/무승부 상태만으로는 결과 화면을 채울 수 없으므로, 각 플레이어의 이름·아바타·순위·완주 시간·별점·승리 조건 결과를 양쪽 클라이언트에 동기화한다.
5. **에셋 정리** — `M2 레이싱 게임 UI 디자인`의 로컬 원본과 안전한 `Assets/Art/Sprites` 에셋을 Unity 스프라이트로 정리해 사용한다. HTML 파일이나 웹 폰트에 런타임으로 의존하지 않는다.

### 검수 및 커밋 기준

- 여섯 화면을 1280×720 기준과 실제 16:9 해상도에서 확인한다. 방 코드, HUD, 버튼, 결과 카드가 겹치거나 읽기 어려워서는 안 된다.
- 호스트와 참가자의 메인·로비·레이스·결과 흐름을 각각 실제 Play 모드에서 확인하고 PlayMode 회귀를 실행한다.
- 각 기능 묶음은 검수 후 명확한 메시지로 별도 커밋한다.

## UI 구현 이후 우선 후속

1. **Relay 2인 라이브 검증** — 인터넷 사용이 허용된 뒤 실제 호스트·클라이언트 환경에서 방 코드, 설정 동기화, 스피드전 무작위 아이템 미생성, 5초 자동 휘발유 지급, 결과/재시작 흐름을 검증한다.
2. **출시 전 정리** — 상태 문서의 오래된 테스트 수치·스케일링 설명을 정리하고, 개발용 스테이지 단축키·히트박스 표시·임시 시각 요소의 제거 또는 개발 빌드 한정을 검토한다.

## 온라인 검증 보류

- Relay 방 코드 실제 호스트·클라이언트 2인 라이브 테스트가 남아 있다.
- 현재 작업 지침의 인터넷 제한 때문에 실제 Cloud 인증·방 생성은 실행하지 않는다. 사용자가 인터넷 사용을 허용한 다음 온라인 플레이테스트에서 수행한다.
