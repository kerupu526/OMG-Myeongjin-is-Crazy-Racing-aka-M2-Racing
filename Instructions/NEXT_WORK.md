## 먼저 할 일 — UI 실제 구현 후속

현재 HUD·방 설정의 기능은 구현됐지만, 다음 범위로 한 번 더 다듬는다.

1. 완료 — `RoomSettingsUI`와 Bootstrap Canvas를 Scale With Screen Size로 통일했다.
2. 완료 — 결과 화면에 순위·완주시간·별점·승리 조건을 실제 데이터로 정리했다.
3. 진행 중 — 브리핑은 중앙 카드로 정리했다. 카운트다운·입력 도움말은 최종 해상도별 실제 화면을 보고 미세 조정한다.
4. 각 단계는 Unity Play 모드 화면 확인과 PlayMode 회귀를 거쳐 별도 커밋한다.
